## Current Controls:
- Move     : WASD   -  Left Joystick
- Camera   : Mouse  -  Right Joystick

- Crouch   : CTRL   -  East-Button
- Interact : F      -  West-Button
- Attack   : LMB    -  Right Trigger
- Navigate Menus : D-Pad
